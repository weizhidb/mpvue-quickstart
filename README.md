# mpvue-quickstart

> fork 自 [vuejs-templates/webpack](https://github.com/vuejs-templates/webpack) 修改而来

## 基本用法

``` bash
$ npm install -g vue-cli
$ vue init F-loat/mpvue-quickstart my-project
$ cd my-project
$ npm install
$ npm run dev
```

## 主要特性

* 移除无效依赖
* 集中配置页面信息
* 合理化目录结构
* 支持新增页面热更新
* 自动清理失效文件
* 配置文件模块化
